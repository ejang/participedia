# 참여 방법 라이브러리 (한국어) – 심플 웹페이지 템플릿

## 목적
- 한국어로 번역된 CSV를 웹페이지에서 **검색/필터/카드 형태**로 쉽게 찾아볼 수 있게 합니다.
- 서버 없이도 사용 가능(정적 페이지).

## 사용 준비
- 이 폴더에 `data.csv`를 넣으면 자동 로드됩니다.
- 또는 웹페이지 상단의 **CSV 불러오기**로 직접 선택할 수 있습니다.

## 가장 좋은 CSV 만들기 (추천)
1) Google Sheets에서 번역이 완료되게 둡니다.
2) `파일 > 다운로드 > 쉼표로 구분된 값(.csv)`로 내보냅니다.
3) 파일명을 `data.csv`로 바꿔 이 폴더에 넣습니다.

⚠️ 번역 컬럼이 `=GOOGLETRANSLATE(...)` 같은 수식 상태면,
웹페이지에는 수식 문자열이 보입니다. 반드시 번역 결과(값)로 내려받은 CSV를 사용하세요.

## 실행 방법
### 로컬(간단)
- `index.html` 더블클릭
- 자동 로드가 안되면 상단에서 CSV를 직접 불러오기

### 로컬(권장: fetch 안정)
- 터미널에서 이 폴더로 이동 후:
  - `python -m http.server 8000`
- 브라우저:
  - `http://localhost:8000`
